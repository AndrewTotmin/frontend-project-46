# Описание

### Приложение для вычисления разницы в данных двух файлов для форматов JSON и YAML

## Установка

### Клонируйте репозиторий:
```
git clone git@github.com:CatIsCode/frontend-project-46.git
```
###  Выполните команду:
```
make install
```
###  Установите локальные пакеты:
```
npm link
```
